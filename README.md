# WittySearch Demo 🔍

This repository contains the static frontend demonstration of **WittySearch**, a specialized file search engine. This demo serves as a portfolio piece showcasing the project's clean, accessible, and responsive user interface design.

⚠️ **Important:** This repository is strictly for the frontend proof-of-concept. For the fully functional Python/Flask backend and the core file system traversal logic, please visit the [Main WittySearch Repository](https://github.com/wittywilla/wittysearch).

## 🚀 About This Demo
The code in this repository powers the live site demo available at the link above. It is designed to serve as a **basic proof of concept** for those interested in exploring the project's UI, layout, and design philosophy without needing to set up a local server environment. 

**Please note:** This demo *does not* utilize the Python backend. It is a static representation meant to showcase the responsive design, accessibility features, and visual components (like the theme toggles and image previews) without the active Flask routing and live file traversal logic running behind the scenes. 

---

## ✨ Frontend Features Showcased
* **Responsive, Mobile-First Design:** Crafted with semantic HTML5 and modern CSS to ensure a smooth, readable experience on any device.
* **Accessibility Focused:** Built with proper ARIA attributes and a "skip to main content" link for improved screen reader navigation and keyboard usability.
* **Theater-Mode Image Previews:** Includes dynamic, theater-style popups for visual search results.
* **Dark/Light Theme Toggle:** Built-in theme switching capability for user comfort.
* **Static Data Representation:** Uses sample JSON structures to simulate search results and demonstrate UI layout.
* **Branded Integration:** Custom WittyWilla favicon integration and persistent GitHub navigation links.

*(To see the robust Python Flask backend and Jinja2 dynamic UI rendering, check out the [main repository](https://github.com/wittywilla/wittysearch).)*

## 🛠️ Demo Technology Stack
* **Frontend:** HTML5 (Semantic), CSS3 (Mobile-First), JavaScript (Theme toggling & theater-mode previews)
* **Data Management:** Static JSON for demonstration purposes
